Setting up:

* Pick "Downloads"
* Pick "Download repository"
* Extract to `%PROGRAMDATA%/GameMakerStudio2/Skins/Yellow`
* Pick the skin via menu:File - Preferences - General - IDE Skin
* Restart the IDE
